# HEARTBEAT.md - 野码AI自动任务

## 每日任务:

**08:00 - 财经分析汇报** (优先级最高)
- 运行 financial_analyzer.py 生成日报
- 分析财经事件、趋势、政策
- 扫描A股和港股通投资机会
- 分析目标公司投资逻辑
- 给出买入/卖出建议
- 输出格式: 📊 [日期] 财经分析
- 数据源: Ashare(A股) + AKShare(港股)

## 每2小时执行(轮换检查):

1. **热点扫描**
   - 检查Hacker News热门
   - 检查GitHub trending
   - 检查NewsNow实时热点: https://newsnow.busiyi.world/c/realtime
   - 查找新机会/痛点/爽点
   - 如发现高价值机会 → 立即制定MVP计划并汇报

2. **项目进度检查**
   - 当前开发项目的进度
   - 遇到的问题和解决方案
   - 是否需要调整方向

3. **GitHub研究** (空闲时)
   - 研究热门项目
   - 总结新最佳实践
   - 记录到TOOLS.md或memory

## 输出格式:

**有新机会/进展时:**
```
🔍 野码AI汇报 [时间]:
发现: [热点/机会]
MVP: [产品概念]
进度: [完成X%]
问题解决: [日志]
需反馈?
```

**无新进展时:**
```
HEARTBEAT_OK
```

## 注意:
- 晚间(23:00-08:00)降低频率,仅紧急情况汇报
- 保持简洁,信息密集
- 记录所有问题和解决方法
