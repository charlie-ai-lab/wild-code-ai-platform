## 🚀 部署指南

### 快速启动（Docker）

```bash
# 克隆仓库
git clone https://github.com/charlie-ai-lab/wild-code-ai-platform.git
cd wild-code-ai-platform

# 启动所有服务
docker-compose up -d

# 访问应用
# 后端: http://localhost:8000
# 前端: http://localhost:3000
# API文档: http://localhost:8000/docs
```

### 手动启动

#### 后端

```bash
cd mvp_project/backend
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python main.py
```

#### 前端

```bash
cd mvp_project/frontend
npm install
npm run dev
```
